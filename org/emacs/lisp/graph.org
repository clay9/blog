* what
  artist-mode和dot都可以完成绘图的功能.

  | mode        | 简述      | 优点                          | 缺点                                      |
  |-------------+-----------+-------------------------------+-------------------------------------------|
  | artist-mode | ASCII绘图 | 1.ASCII表示图形               | 1.功能少                                  |
  |             |           | 2.短小精悍                    | 2.需要手动绘制图形                        |
  |-------------+-----------+-------------------------------+-------------------------------------------|
  | mermaid     | dot绘图   | 1.只关注逻辑设计,布局自动生成 | 1.生成的为图片文件, 而非可嵌入的ASCII代码 |
  |             |           |                               | 2.需要学习dot语言                         |
  |-------------+-----------+-------------------------------+-------------------------------------------|
