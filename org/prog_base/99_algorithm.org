** 什么是
   数据结构分为
   - 物理结构
     即在内存中的结构. 有顺序存储(内存连续) 和 链式存储(内存可不连续)
   - 逻辑结构
     线性关系, 一对一, 一对多, 多对多
** 逻辑结构
*** 线性关系
    数组, 链表
    栈, 队列
*** 一对一
    hash table
*** 一对多
    - tree 树
    - binary tree 二叉树
    - search binary tree 搜索二叉树
    - blance binary tree 平衡二叉树

* 搜索算法
  线性查找: 遍历. 线性查找可以进化为哈希查找来降低时间复杂度
  二分查找: 数组, 有序 log(n)
  哈希查找: 借助额外的hash table
