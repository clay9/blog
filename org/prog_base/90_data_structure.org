* 概述|总结
** 什么是
   数据结构分为
   - 物理结构
     即在内存中的结构. 有顺序存储(内存连续) 和 链式存储(内存可不连续)
   - 逻辑结构
     线性关系, 一对一, 一对多, 多对多
** 逻辑结构
*** 线性关系
    数组, 链表
    栈, 队列
*** 一对一
    hash table
*** 一对多
    - tree 树
    - binary tree 二叉树
    - search binary tree 搜索二叉树
    - blance binary tree 平衡二叉树
    - complet binary tree 完全二叉树
    - 堆

    | name                | 特征                                         | 备注 |
    |---------------------+----------------------------------------------+------|
    | tree                |                                              |      |
    |---------------------+----------------------------------------------+------|
    | binary tree         | 1.任意节点 叶度 <=2                          |      |
    |---------------------+----------------------------------------------+------|
    | search bianry tree  | 1.binary tree                                |      |
