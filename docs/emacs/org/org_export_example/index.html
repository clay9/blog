    Internal Links #

    1. “\[\[#FirstLink\]\]” Entry with “CUSTOM_ID”
    2. “\[\[*FirstLink\]\]” FirstLink
    3. “\[\[FirstLink\]\]”
    4. “\[\[SecondLink\]\]”
    [[#target]] 只会找 Entry with CUSTOM_ID为target
[[*target]] 只会找 Headline name为target
[[target]] 优先找 entry with <<target>>; 其次Name为target
