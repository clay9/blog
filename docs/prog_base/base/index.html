    lambda演算 #

    lambda演算教程 (机翻版)

    λ 演算中的中心概念是“表达式”。
    一个表达式可以只是一个名称，也可以是一个函数。

    “名称”是一个标识符，对于我们的目的，可以是任何字母 a、b、c 等。
    函数使用希腊字母 λ 来标记函数参数的名称。函数的“主体”指定如何重新排列参数

    < 表达式 > := < 名称 > | < 函数 > | < 应用 >
    < 函数 > := λ < 名称 > . < 表达式 >
    < 应用 > := < 表达式 >< 表达式 >
