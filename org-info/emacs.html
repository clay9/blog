1 emacs基础操作

1.1 Helm

Helm改善了用户"查找"时候的体验; helm与传统方式 对比:

  • Helm的过滤方式为 正则表达式
  • 传统的过滤方式为 首字母扩展

1.2 org mode

org-mode一直被称为神器, 据传很多人为了它转到了emacs.

1.3 eshell

eshell是emacs下非常好用shell; 主要优点:

  • 与elisp的无缝结合

  • 语法特性方面比bash shell更有意思 

    cd /ssh:clay@192.168.0.97:~

1.4 画图

emacs支持 ASCII图形 和 dot图形等

1.5 快捷键

emacs按键与vim按键 个人理解:

  • emacs功能强大, 却对手指不友好, 更适合在编辑的时候使用
  • vim 功能缺少, 对手指友好, 更适合在查看文档的时候使用

1.6 elisp

elisp 是emacs下的lisp

3 emacs奇淫技巧

3.1 figlet

把字符艺术化

3.1.1 安装

  1. bash中安装back-ends 

    brew install figlet

  2. emacs中安装front-ends 

    package-list-package RET figlet RET

3.1.2 配置

  1. figlet默认路径 

    setq figlet-default-directory "/usr/local/bin"

  2. 默认艺术字体 

    setq figlet-default-font "banner3-D"

    已安装的字体可以在bash中通过showfigfonts查看

3.1.3 使用

M-x figlet RET <string> RET
