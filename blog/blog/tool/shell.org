     1. exit 与 return
	return 退出函数; 在非函数地方, 无效
	exit 在任何地方, 都代表推出sh

   2. $# 代表入参
	sh脚本的入参在非函数地方调用
	函数的入参在调用函数的地方传递

   3. $@ 可以传递所有的参数到下一层

   4. 数组使用echo打印的话 只会显示第一个

   5. [ ] 与 [[ ]] 区别
	[[ ]]更高级, 识别与或非等; 而[ ] 不识别
	
   6. ' ' 与 " "区别 
	在 ' '中的变量不会被展开; '$a'显示出来是字面值$a
